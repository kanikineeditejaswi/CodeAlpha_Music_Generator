# 🎵 Sakura Melody AI

A web-based music generation application that creates unique melody sequences and enables real-time playback through an interactive and visually engaging interface.

## 🚀 Project Overview

This project was developed to demonstrate how technology can be used to create engaging musical experiences through automated melody generation and real-time audio playback.

## ✨ Key Features

* AI-inspired melody generation
* Random musical note sequence creation
* Real-time melody playback
* Music stop functionality
* Interactive and responsive user interface
* Animated music visualizations
* Modern glassmorphism design
* Tone.js audio integration

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Tone.js

## ⚙️ How It Works

1. Open the application in a web browser.
2. Click the **Generate Melody** button.
3. A unique melody sequence is generated automatically.
4. Click the **Play** button to listen to the generated melody.
5. Use the **Stop** button to stop playback at any time.
